## 一、命题用途枚举

基于国内 K12 教学实践，枚举以下用途分类（参考北京/上海/深圳教研室常用命题规范）：

| 一级分类 | 二级用途 | 题量建议 | 难度范围 | 超时限制 |
|----------|----------|----------|----------|----------|
| **日常作业** | 课堂练习 | 3-5题 | L1基础 | 5-8分钟 |
| | 课后巩固 | 5-10题 | L1→L2 | 15-20分钟 |
| | 周末作业 | 15-20题 | L1→L2 | 30分钟 |
| **阶段测评** | 单元测验 | 15-20题 | L1→L2 | 40分钟 |
| | 月考（月度检测） | 20-25题 | L1→L3 | 60分钟 |
| | 期中考试 | 25-30题 | L1→L3 | 90分钟 |
| | 期末考试 | 30-35题 | L1→L3 | 90分钟 |
| **专项训练** | 假期作业（寒假/暑假） | 30-50题 | L1→L2 | 不限 |
| | 薄弱点专练 | 10-15题 | 自适应 | 20分钟 |
| | 竞赛预备（奥数/英语竞赛） | 10-15题 | L3→L4 | 40分钟 |
| | 小升初衔接 | 15-20题 | L2→L3 | 40分钟 |

> 数据来源：参考《义务教育课程标准（2022版）》评价建议 + 北上广深名校命题实践

## 三、题型扩展

### 3.1 基础题型（已有）✅
选择题、填空题、计算题、应用题

### 3.2 进阶题型（新增）
| 题型 | 适用学科 | 难度 | 说明 |
|------|----------|------|------|
| 判断题 | 语/数/英 | L1-L2 | 快速判断对错 |
| 匹配题 | 语文/英语 | L1-L2 | 连线/拖拽匹配 |
| 完形填空 | 语文/英语 | L2-L3 | 篇章理解 |
| 阅读理解 | 语文/英语 | L2-L4 | 短文+多题 |
| 解答题 | 数学 | L2-L4 | 写出完整过程 |
| 作图题 | 数学 | L3 | 几何绘图 |
| 写作题 | 语文/英语 | L3-L4 | 命题作文 |

### 3.3 奥数/竞赛题型
| 题型 | 适用年级 | 说明 |
|------|----------|------|
| 逻辑推理 | 3-6年级 | 鸽巢原理/逻辑谜题 |
| 数论初步 | 4-6年级 | 质数/因数/整除 |
| 行程问题 | 3-6年级 | 相遇/追及/环形 |
| 工程问题 | 5-6年级 | 工作效率/合作 |
| 浓度问题 | 6年级 | 配比/混合 |
| 几何计数 | 3-6年级 | 三角形/线段计数 |
| 数列规律 | 3-6年级 | 等差数列/斐波那契 |
| 抽屉原理 | 3-6年级 | 鸽巢应用 |

## 四、知识图谱可视化设计

### 4.1 数据模型
```
知识点节点 {
  id, name,          // "分数加减法"
  subject,           // 数学
  grade,             // 四年级
  curriculum_code,   // 课标编号 "C-M-4-2-3"
  difficulty,        // L1|L2|L3|L4
  prerequisites[],   // 前置知识点ID
  next[],            // 后续知识点ID
  related[],         // 相关知识ID（跨章节关联）
}
```

### 4.2 视图模式（切换）

| 模式 | 可视化 | 交互 |
|------|--------|------|
| **课标维度** | 森林图 - 按学段→年级→单元→知识点四层树形展开 | 点击展开/收起，搜索高亮路径 |
| **难度维度** | 雷达图 - 6维（计算/推理/应用/记忆/理解/创新） | 拖拽滑块调整难度权重，图谱实时变色 |
| **关联维度** | 力导向图 - 节点=知识点，连线=前置/后置关系 | 拖拽节点，hover显示详情，点击选中并高亮关联链路 |
| **时间进度** | 甘特图 - 按学期周次排列知识点 | 滑块选择当前周次，自动定位 |

### 4.3 交互设计

| 动作 | 响应 |
|------|------|
| 点击知识点节点 | 弹出详情卡片（课标编号/所属年级/题量/掌握度） |
| 点击"选它为考点" | 自动填入搜索框 |
| 拖拽节点 | 力导向图中自由拖动 |
| 搜索框输入 | 图谱中匹配节点高亮 + 自动滚动到视野 |
| 底部滑块 | 调节难度权重（1-5），图谱节点大小/颜色深浅变化 |
| 模式切换 | 顶部4个Tab切换视图 |
| 双击空白区 | 重置缩放/位置 |
| 鼠标滚轮 | 缩放 |

### 4.4 默认状态
- 打开出题页 → 知识图谱默认显示**课标维度**
- 根节点 = 当前年级 + 当前学科
- 已对标的知识点默认展开并高亮（蓝色边框）
- 当前选中的知识点节点放大+呼吸动画

### 4.5 布局
```
┌──────────────────────────────────────────┐
│  图谱模式: [课标] [难度] [关联] [进度]    │
├────────────────────┬─────────────────────┤
│                    │  已选考点:           │
│   知识图谱可视化    │  □ 分数意义          │
│   (自适应高度)      │  □ 同分母加减法      │
│                    │  □ 异分母加减法      │
│                    │  [全部清除] [开始出题]│
│                    │                     │
│                    │  难度滑块: ─○───     │
├────────────────────┴─────────────────────┤
│  [搜索知识点...]  年级:[▼] 学科:[▼]       │
└──────────────────────────────────────────┘
```
