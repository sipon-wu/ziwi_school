# 变更需求文档：全局教学上下文同步修复

> 日期：2026-06-22  
> 作者：老P  
> 状态：已编码 · 已构建 · 待测试  

---

## 1. 背景

发现 `TopNavBar` 头部切换学科/年级/教材后，子页面的下拉框缺省值不跟随变化。根因是 **TeachingContext（全局教学上下文）** 与页面本地 state 之间缺乏正确的单向数据流绑定。

## 2. 影响范围

| 组件 | 问题等级 | 现象 |
|------|---------|------|
| ExerciseGenerator | P0 | 学科/年级下拉框不跟随头部 |
| SettingsPage | P0 | 设置页保存后头部不更新，设置页初始化不读全局值 |
| LessonPlanEditor | P1 | 新建教案默认硬编码"语文·四年级" |
| KnowledgeGraph | P2 | 初中知识点出现在小学阶段图谱中 |

## 3. 变更清单

### 3.1 ExerciseGenerator.tsx — 移除本地 state，直接读写 TeachingContext

**文件**: `code/frontend/src/pages/ExerciseGenerator.tsx`

**修改前**:
```ts
const [subject, setSubject] = useState(teaching.subject)
const [grade, setGrade] = useState(...)
// 4个 useEffect 做双向同步（竞态风险）
useEffect(() => { setSubject(teaching.subject) }, [teaching.subject])
useEffect(() => { setGrade(...) }, [teaching.grade])
useEffect(() => { teaching.setSubject(subject) }, [subject])
useEffect(() => { teaching.setGrade(...) }, [grade])
```

**修改后**:
```ts
// 直接消费 TeachingContext，无本地 state
const gradeName = GRADE_NAMES[teaching.grade - 1] || '四年级'
// 下拉框: value={teaching.subject} onChange={e => teaching.setSubject(...)}
// 年级框: value={gradeName} onChange={e => teaching.setGrade(idx+1)}
// TeachingContext 变更时自动更新难度/题量/题型等推荐值
```

### 3.2 SettingsPage.tsx — 接入 TeachingContext

**文件**: `code/frontend/src/pages/SettingsPage.tsx`

**修改内容**:
- 新增 `import { useTeaching, ALL_MATH_TEXTBOOKS, ALL_ENGLISH_TEXTBOOKS }`
- `grade`/`subject` 初始化从 `teaching.grade`/`teaching.subject` 读取
- `textbookSubject`/`textbookVersion` 初始化从 TeachingContext 读取
- `handleSave` 保存时同步回 `teaching.setSubject()`/`teaching.setGrade()`
- 教材版本确认按钮同步回 `teaching.setTextbookMath()`/`teaching.setTextbookEnglish()`
- 教材版本下拉框按学科联动（数学→多版本，英语→PEP/外研版等，语文→部编版）
- 年级下拉框扩展到一到九年级

### 3.3 LessonPlanEditor.tsx — 从 TeachingContext 初始化

**文件**: `code/frontend/src/pages/LessonPlanEditor.tsx`

**修改内容**:
- 新增 `import { useTeaching }`，初始化取 `teaching.subject`/`teaching.grade`
- 新建模式下 onChange 同步回 TeachingContext（编辑已有教案不覆盖）

### 3.4 KnowledgeGraph.tsx — 按年级/学期边界过滤

**文件**: `code/frontend/src/components/KnowledgeGraph.tsx`  
**文件**: `code/frontend/src/pages/ExerciseGenerator.tsx`（传参调用）

**新增 Props**: `grade?: number`, `semester?: string`

**过滤规则**:
```
学科 = 当前学科
年级 ≤ 当前年级
  ├── 年级 < 当前年级 → 全部显示（已学过的知识体系）
  └── 年级 = 当前年级 → 学期 ≤ 当前学期（只看已学内容）
超过当前年级的知识点 → 不显示
```

**效果示例**: 数学·4年级·下学期 → 图谱只显示 1-3年级全部 + 4年级上+下学期，5-6年级不出现。
