## 第二幕: AI 教案生成 (3分钟)

### 串讲词
```
【点击侧栏"教案备课" → 点击"新建教案"】

现在我们进入最核心的场景——备课。张老师今天要讲"小数加减法"，
她没有现成教案，以往需要花1-2小时搜索整理。

现在她只需要：选择学科→数学，年级→四年级，输入课题名称→"小数加减法"，
选择核心素养模板，点击"AI 生成教案"。

【等待15-30秒，展示生成结果】

看，一份完整的教案生成了。这不是简单的模板填空——
系统背后的AI完成了以下工作：
1. 检索课标数据库，找到对应的课标条目(4.1.4)
2. 根据选定的核心素养模板生成结构化教案
3. 教学目标、重难点、教学过程、板书设计一应俱全
4. 右上角显示"课标对齐: 1/1项"

这份教案可以直接保存，也可以手动编辑调整。张老师检查无误后点击"定稿并保存"，
教案就锁定了。
```

### 演示操作
- 侧栏 → 教案备课 → 新建教案
- 学科选"数学"，年级选"四年级"
- 输入课题"小数加减法"
- 点击"AI 生成教案"
- 展示教案内容
- 点击"定稿并保存"

### 亮点
- 课标自动对齐(RAG检索)
- 多模板支持(核心素养/三维目标/单元教学)
- 编辑回填(已有教案可二次编辑)
- 确认弹窗防止误操作

## 第三幕: 知识图谱出题 (4分钟)

### 串讲词
```
【点击侧栏"出题组卷" → "新建出题"]

接下来是出题环节，这是今天展示的核心亮点——知识点图谱。

请注意顶部，系统已经自动推荐了：
- 学科：数学(来自教学配置)
- 教材：人教版(来自学校锁定)
- 当前进度：四年级下册，第三单元"运算定律"

张老师不需要手动输入这些信息，系统全部帮她想好了。

往下看，命题用途有8种——从课堂练习到奥数拓展。AI会根据今天是星期一，
智能推荐"课后作业"作为默认用途，自动推荐题量8题、难度L2。

再往下，题型选择器覆盖了基础题型(选择/填空/计算)和进阶题型(判断题/解答题/作图题)。
选择题型后点击"展开知识图谱"。

【点击展开知识图谱】

这就是我们的知识图谱。每个圆点代表一个知识点，蓝色的是当前教材覆盖的。
它有三种布局——树状、环状、网状——现在展示的是树状布局。

点击节点可以选中为出题范围，右侧能看到已选考点列表。
张老师勾选了"小数的意义和性质"和"小数加减法"几个节点后，
点击"用选中考点出题"。

同时她可以设置名校风格——比如选择"人大附中"，系统会模拟该校的命题风格。
然后点击"AI 智能出题"。

【等待生成，展示题目列表】

题目生成完毕。难度配比自动均衡，题型覆盖了选择、填空、计算。
张老师审核无误后就可以发布到班级了。
```

### 演示操作
- 进入新建出题页
- 指出顶部自动推荐的教学进度
- 点击命题用途卡片切换，展示自动推荐变化
- 选择"课堂练习"
- 切换题型选择
- 展开知识图谱
- 切换树状/环状/网状布局
- 切换知识/能力/难度/课标维度着色
- 拖拽难度滑块过滤
- 点击节点选中出题范围
- 选择名校风格"人大附中"
- 点击AI出题

### 亮点
- 知识图谱三种布局 + 四种维度切换
- 命题用途AI智能推荐
- 8种命题用途覆盖完整教学场景
- 名校风格模拟(6所)
- 教材版本区域匹配
- 难度滑块实时过滤

## 第四幕: 批阅管理 (3分钟)

### 串讲词
```
【点击侧栏"批阅管理"】

教案有了，题目出了，学生提交了作业。接下来张老师要批改。

批阅工作台展示了统计概览——12份待批阅，其中3份是AI把握不大的，
用红色高亮标记出来。

点击"进入批阅"，可以看到每一道题的AI批阅结果：
- AI自动评分、给出反馈
- 每一题都有置信度——置信度低于85%的会特别标记为"需要教师复核"
- 教师可以调整分数、添加批注

张老师检查无误后，点击"确认全部批阅"，批阅结果就生效了，
学生端会同步看到。

整个过程，AI完成了80%的批阅工作，教师只需要复核低置信度的部分。
```

### 演示操作
- 侧栏 → 批阅管理
- 指出统计卡片和低置信度区域
- 点击"进入批阅"
- 展示AI评分、置信度、反馈
- 调整一道题的分数
- 点击"确认全部批阅"

### 亮点
- AI自动批阅 + 置信度标注
- 低置信度自动标记为教师复核
- 教师可调整分数和添加批注
- 批阅结果与教案/出题联动

## 第五幕: 多角色闭环展示 (4分钟)

### 5.1 学生端 (1.5分钟)

#### 串讲词
```
【打开新标签页，模拟手机视图访问 /m/student】

知微不只服务教师。现在我们切换到学生视角——四年级的李明同学。

登录后他看到自己的作业列表，上面标注了"待完成"和"已完成"。
我们点进一份数学作业。

答题界面支持四种题型——选择题点选、填空题输入、计算题手写、
还有英语的写作游戏。答完提交后，AI会立即批阅。

批阅结果页展示每一题的对错、正确答案对比，
还有"弱项提示"——系统自动分析他需要加强哪些知识点。

错题本功能把历史错题汇总在一起，可以按学科筛选，
每道题都能复习回顾，标记"已掌握"后移出。
```

#### 演示操作
- 新标签页 → /m/student
- 展示作业列表(待完成/已完成Tab)
- 点击进入答题页
- 展示选择题和填空题交互
- 提交后展示批阅结果页
- 切换到错题本

#### 亮点
- 四种题型在线作答
- AI即时批阅反馈
- 弱项分析 + 错题本
